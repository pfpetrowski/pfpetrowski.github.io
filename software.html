Software

OhtaDStats

OhtaDStats is an R package that contains a variety of tools for the computation of the D statistics derived by Tomoka Ohta in her 1982 paper. It is designed to be highly scalable to process enormous genomic datasets on high throughput computing platforms. It is available on GitHub and downloadable from CRAN. Our manuscript detailing this package is available here.

rsnell

rsnell is an R implementation of the Snell transformation. It generates continuous scores from ordinal measurements. It is available on GitHub and published on CRAN.
